Car Dealing
Website

A car-listing site built in pure HTML with a user-friendly search form and clean table layouts. Designed so visitors could quickly browse vehicles by make, model, year, price, and mileage — then reach out to the dealer in a click.

Year
June 2020
Role
Creator · Designer · Developer
Type
Static website
Status
Completed

Overview

My first real website project — a digital storefront for a car dealership where users could explore listings and find what they were looking for fast. The whole thing was built in plain HTML, which forced me to really understand the building blocks before reaching for any frameworks.

The focus was usability: forms that didn't get in the way, listings that were easy to scan, and links that did exactly what you'd expect.

Features

  • User-friendly search form built from input fields, dropdown menus, and radio buttons.
  • HTML tables displaying listings by make, model, year, price, and mileage.
  • Interactive links that opened detailed listings or contact options.
  • Clear, scannable layout designed around the way people actually shop for cars.

Tech Stack

HTML Forms Tables Semantic Markup

A small site, but it taught me the value of fundamentals. Knowing what HTML can do on its own — and where you actually need more — has shaped how I approach every project since.
