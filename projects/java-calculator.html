Java Calculator

Overview

This was one of my first real desktop applications — a familiar tool, built from scratch in Java to get comfortable with how GUIs actually come together. The point wasn't to make a calculator. The point was to learn how a user's tap on a button turns into code that runs.

NetBeans' drag-and-drop interface builder handled the layout side, which let me focus on the more interesting part: event handling — making sure the right thing happened when the right button got pressed.

Features

  • Graphical user interface laid out using NetBeans' visual builder.
  • Buttons, text fields, and labels arranged for natural calculator use.
  • Event handlers triggered on user interaction with each button.
  • Basic arithmetic — addition, subtraction, multiplication, and division.
  • Clean Java code organized for readability and easy extension.

Tech Stack

Java NetBeans JDK GUI Event Handling

A small project, but one that made event-driven programming click for me. Once you see how user input flows through handlers and back into the UI, you start seeing every app that way.
