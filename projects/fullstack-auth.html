Overview

This project was the culmination of my web development work — taking everything from sessions to databases to security and pulling it together into a working site where users could actually have accounts.

The site handles the full lifecycle of a user account: signing up, signing back in, signing out, and tracking what happens along the way with logging. Behind the scenes, MongoDB stores user records and bcrypt.js makes sure passwords are never readable, even if the database were exposed.

Features

  • User registration with form validation and persistent records in MongoDB.
  • Login and logout flows using client sessions for stateful authentication.
  • Activity logging that tracks user actions for visibility and debugging.
  • Password security using bcrypt.js hashing instead of plain-text storage.
  • Clean front-end built with CSS and JavaScript, designed for clarity.

Tech Stack

JavaScript CSS MongoDB bcrypt.js Sessions Node.js

This project gave me hands-on experience with the security side of full-stack development — understanding why password hashing matters, how session state actually works under the hood, and how to model persistent user data.
